# 🌏 춘천 여행 추천 서비스

## 📌 소개

춘천 여행지를 추천해주는 웹 프로젝트입니다.

## 🚀 기능

* 여행지 추천
* 카테고리별 필터링
* 각자의 성향에 맞게 관광지를 추천해주는 기능입니다.

## 🛠️ 기술 스택

* HTML / CSS / JavaScript


## ⚙️ 실행 방법

```bash
git clone https://github.com/suin-it/recommend-chuncheon-tour.git
cd recommend-chuncheon-tour
# 실행 방법 추가
```
